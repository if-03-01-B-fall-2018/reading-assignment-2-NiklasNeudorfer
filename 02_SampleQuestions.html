HTML

1. Name the components of html elements properly:

  • text header: defines the heading < h1> < h2> ... < h6>
  • paragraph: defines a paragraph
  • html: element defines the whole document
  • body: defines the body of the document

3. Identify void elements and how they are denoted:

  • HTML elements with no content are called empty elements. bsp: < br>

3. Identify attributes:

  • Attributes provide additional information about HTML elements.
  • All HTML elements can have attributes
  • Attributes are always specified in the start tag

4. Write down the correct DOCTYPE declaration for html 5:

  • < !DOCTYPE html>

5. Write down syntactically correct statements fot the following elements:

  • Article (<article>): <article><h1>sample</h1></article>
  • Body (<body>): <body><h1>sample</h1></body>
  • line break (<br/>): <p>sample<br>sample</p>
  • Headings (<h1>, ...): <h1>sample</h1>
  • Section with meta-information (<head>): <head><title>This is a title</title></head>
  • Top level element (<html>): <html><head></head></html>
  • Image named Team.jpg (<img src="Team.jpg">): <img src="Team.jpg">
  • Paragraph (<p>): <p>sample</p>
  • Section (<section>): <section><h1>sample</h1></section>
  • Label appearing in the title bar (<title>): <title>sample</title>
  • Link to other pages (<a>): <a href="https://www.google.at">
  • Comment: <!-- comment -->
  • Head with title and meta elements: <head><title>sample</title><meta charset= "utf-8"></style></head>

CSS

1. Name the components of a css rule properly:

  • A CSS rule-set consists of a selector and a declaration block. (h1{color:red;font-size:12px;})
    property:value;property:value;

2. Use combinators properly: directly child, decendant, adjacent sibling, sibling:

  • directly child selector: >
  • descendant selector: space
  • adjacent sibling selector: +
  • general sibling selector: ~

3. Specify colors by color names and in hexadecimal notation:

  • Tomato / #ff6347

4. Declaration for color, background color:

  • font color: < h1 style="color:Tomato;">Hello World
  • background color: < h1 style="background-color:Tomato;">Tomato

5. Pseudo classes for the < a> element:

  • a:link,a:visited,a:hover,a:active

6. Declaration for background image:

  • body{ background-image: url("02_Background.jpg"); }

7. Declaration fot text allignment (left, right, center, justify)

  • h1{ text-align: center; }

8. Declaration for text decoration (overline, underline, line-through)

  • h1{ text-decoration: underline; }

9. Declaration for text transformation (uppercase, lowercase, capitalize)

  • p.uppercase{ text-transform: uppercase; }

10. Identify the difference between serif and sans-serif fonts

  • Serif fonts have that extra stroke or decorative design on the end of letters.Whereas Sans-Serif doesn't have any such design or stroke.

11. Declarations for font families

  • generic family - a group of font families with a similar look (like "Serif" or "Monospace")
  • font family - a specific font family (like "Times New Roman" or "Arial")
  • Example:
    p { font-family: "Times New Roman", Times, serif}

12. Declarations for font style normal and italic

  • p.italic{ font-style: italic; }

13. Declarations for font sizes

  • h1 { font-size: 49px; }

14. Declarations for font weights

  • p.thick{ font-weight: bold; }
